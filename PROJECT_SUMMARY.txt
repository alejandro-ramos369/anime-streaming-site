                        TECHNOLOGY STACK
================================================================================

Frontend:
  - React 19 with TypeScript
  - TailwindCSS 4 for styling
  - Vite for fast builds
  - Axios for API requests

Data Source:
  - AniList GraphQL API (official anime database)
  - No authentication required

Deployment:
  - Vercel (recommended)
  - Netlify
  - GitHub Pages
  - AWS S3 + CloudFront
  - Firebase Hosting

================================================================================
                        DEPLOYMENT OPTIONS
================================================================================

1. VERCEL (Recommended)
   - Sign up at https://vercel.com
   - Connect GitHub repository
   - Click Deploy
   - URL: https://anime-streaming-site.vercel.app
   - Time: 2 minutes

2. NETLIFY
   - Sign up at https://netlify.com
   - Connect GitHub repository
   - Click Deploy site
   - URL: https://anime-streaming-site.netlify.app
   - Time: 2 minutes

3. GITHUB PAGES
   - Update vite.config.ts with base path
   - Run: npm run build
   - Push to GitHub
   - Enable in repository settings
   - URL: https://yourusername.github.io/anime-streaming-site
   - Time: 5 minutes

4. AWS S3 + CloudFront
   - Create S3 bucket
   - Upload dist/ files
   - Create CloudFront distribution
   - Time: 10 minutes

5. FIREBASE HOSTING
   - Install Firebase CLI
   - Run: firebase init hosting
   - Run: firebase deploy
   - Time: 5 minutes
